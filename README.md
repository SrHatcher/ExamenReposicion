# Instrucciones de instalación

Sigue estos pasos para configurar y ejecutar el proyecto:

## 1. Clonar el repositorio

```bash
git clone <URL_DEL_REPOSITORIO>
```

## 2. Instalar dependencias
```bash
npm i
```

## 3. Ejecutar el proyecto
```bash
npm run start
```
